## **1. 개요**

---

### **프로젝트명**

- 인트라넷 서비스 개발

### **목표**

- 직원 데이터베이스 구축 및 관리
- 근태 관리 시스템 구축
- 관리자의 정보조회,생성,수정,삭제 기능
- 사용자의 정보조회 및 수정 기능
- 기업 내 행사 사진 공유
- 사용자 친화적인 웹 인터페이스 제공

### **사용자 정의**

- 인트라넷 관리자
- 인트라넷 사용자

## **3. 시스템 아키텍처**

---

### **아키텍처 개요**

- 프론트엔드: JavaScript, HTML, CSS, Axios.js, Vite
- 백엔드: Node.js (Express.js)
- 데이터베이스: Sqlite3

### **모듈 구성**

- 프론트엔드: 사용자 인터페이스, 데이터 시각화, API 연동
- 백엔드: RESTful API, 비즈니스 로직, 데이터베이스 연동
- 데이터베이스: 직원 개인 정보, 근태 정보
